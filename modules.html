publish_data_product(catalog_id: str, data_product_id: str, workspace_id: str | None = None, collection_id: str | None = None, config: MtnaRdsPostmanPublisherConfig | None = None) → str

Publish a data product as a collection under an existing workspace.

If collection_id is specified, its content will be replaced (but the collection id remains the same). If both workspace_id and collection_id are provided, collection_id takes precedence, and the collection will be replaced. Otherwise, workspace_id must be specified and a new collection will be created.
